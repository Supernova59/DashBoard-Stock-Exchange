# DashBoard Boursier

Un tableau de bord financier haute performance permettant de surveiller en temps réel (20m de delai) les actifs boursiers.

## Installation
1. Clonez le dépôt.
2. Modifier le fichier `config.txt` à la racine du projet, collez uniquement votre clé API Alpha Vantage à l'intérieur.
3. Lancez via un serveur local (ex: Live Server sur VS Code).
